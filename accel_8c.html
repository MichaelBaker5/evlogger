Detailed Description

Handles configuration and initialisation of the CMA3000 onboard accelerometer.

This module is based on the TI example code (see the notice in the source code) buit is heavily modified to use a finite state machine (FSM) type approach to getting data from the accelerometer such that very little CPU time is required (since the logger is typically busy with other things).
